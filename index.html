2048 Game

This is a classic 2048 game that is free and with no ads.

How to Play:

  • Use your arrow keys (or swipe on mobile) to move the tiles.
  • When two tiles with the same number touch, they merge into one!
  • The goal is to reach the 2048 tile.
  • Keep merging tiles to get higher numbers like 4, 8, 16, 32, 64, 128, 256, 512, 1024, and finally 2048.
  • The game ends when the board is full and there are no more possible moves.

Good luck and have fun!

This game is almost entirely vibe-coded in Gemini Code. If you find any issues, feel free to report them.
